This is a 2 player game in which 2 players can play using one keyBoard

Controls : 
Player1 : Four arrow keys for movement & Right control key to spawn Bomb
Player2 : ASDW keys for movement & Left control key to spawn Bomb

Even if one player died (either by enemy or by bomb), the game will be restarted automatically.

'Scene01' in 'Scenes' folder is the scene to open the game.
